## Klinik Vue.js Frontend

### Overview
The **Klinik Vue.js Frontend** is the user interface for the Klinik system, a complete clinic management solution. Developed with **Vue.js 3**, this frontend application integrates smoothly with the **Klinik Laravel API**, providing a responsive and user-friendly experience to manage clinic operations like appointments, schedules, patient records, and more.

### Features
- 🩺 **Appointment Scheduling**: A dynamic calendar and forms for scheduling and managing appointments.
- 🩺 **Patient Management**: Convenient access to patient information and medical history.
- 👨‍⚕️ **Doctor Dashboard**: A personalized dashboard for managing doctor schedules and appointments.
- 📊 **Reports and Statistics**: Visual representation of clinic operations to assist with decision-making.
- 🌙 **Dark Mode**: Toggle between light and dark themes for enhanced user experience.
- 📱 **Responsive Design**: A fully responsive layout for easy management of clinic tasks on any device.
- and more...

### Tech Stack
- **Framework**: Vue.js 3
- **State Management**: Pinia
- **UI Components**: Tailwind CSS, DaisyUI, PrimeVue
- **Calendar**: Vue Cal
- **Notifications**: Vue Sonner
- **Charts**: PrimeVue Charts
- **API Communication**: Axios
- **Routing**: Vue Router 4
